# 基于大数据的租房推荐系统 python535

## 项目概述

本项目是基于大数据的租房信息推荐系统，以解决用户在海量租房信息中难以快速准确找到合适房源的问题。通过运用先进的数据爬取、处理、推荐算法和可视化技术，为用户提供全面、准确、个性化的租房信息。

## 技术栈

- 前端：Vue.js
- 后端：Django
- 数据爬取：Scrapy
- 数据库：MySQL
- 数据可视化：Echarts
- 数据分析：Python（包括pandas等库）

## 功能模块

### 数据爬取与清洗

- 使用Scrapy框架进行数据爬取。
- 通过XPath和正则表达式对数据进行解析和清洗。

### 数据库设计与管理

- 使用MySQL存储数据，通过Django ORM进行操作。
- 表结构包括租房信息、用户信息、搜索关键词、用户搜索记录等。

### 推荐算法

- 实现基于协同过滤的推荐算法。
- 结合用户行为和喜好进行个性化推荐。

### 前后端架构

- 前端使用Vue.js实现交互效果。
- 后端使用Django构建逻辑，并通过RESTful API与前端交互。

### 可视化展示

- 利用Echarts进行数据可视化。
- 将推荐结果以图表形式显示在大屏幕上。

### 统计与分析

- 对租房信息进行多维度统计和分析。
- 使用Python数据分析库进行数据处理。

## 系统角色

- 普通用户：搜索、查看推荐房源等。
- 系统管理员：管理租房信息、用户数据等。

## 运行环境

- 前端：现代浏览器
- 后端：Python环境，推荐使用Django官方推荐环境配置

## 部署步骤

1. 搭建开发环境（Python、Django、Vue.js等）。
2. 数据库MySQL的配置和部署。
3. Scrapy爬虫框架的安装和配置。
4. 前后端代码的编译和部署。
5. Echarts可视化工具的集成。
